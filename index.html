Cloud setup guide (Firebase)

Use this quick checklist to connect FineLedger to your own cloud safely.

  1. Create a Firebase project at console.firebase.google.com.
  2. Enable Authentication → Google in your Firebase project.
  3. Create a Firestore Database and start in production mode.
  4. In Project Settings, open your Web App config and copy these values: API Key, Auth Domain, Project ID, Storage Bucket, Messaging Sender ID, and App ID.
  5. In FineLedger, paste those values into Firebase Cloud Connection.
  6. Choose a Workspace ID (letters, numbers, dash, underscore), for example my-team-ledger.
  7. Click Save & Connect Cloud, then click Sign in with Google.
  8. After sign-in, confirm the top cloud indicator says Cloud sync active.
  9. Optional: Use Clear Saved Config if you are on a shared computer.
Tip: Keep Firebase API values in your own project only. This app is designed so each person can bring their own cloud setup.
